myFlix Documentation

Web application that provides access to information about different movies, directors, and genres.
Users are able to sign up, update their personal information, and create a list of their favourite movies.

Request URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, containing a title, description, genre, director and imageUrl properties. Example:
{title: 'Inglourious Basterds', description: '#', genre: { name: 'Drama', description: '#', }, director: { name: 'Quentin Tarantino', bio: '#', }, imageUrl: '#'}
Get data about a genre
(description) by name/title		 /movies/genre/[genreName] GET None A JSON object holding data about a single genre, containing a name and a description property. Example:
genre: { name: 'Drama', description: '#'}
Get data about a director
(bio, birth year, death year) by name		 /movies/director/[directorName] GET None A JSON object holding data about a single director, containing a name and a bio property. Example:
director: { name: 'Quentin Tarantino', bio: '#'}
Add new user /users POST A JSON object holding data about the user to add structured like:
{id: 1, name: 'Kim', favouriteMovies: []} 		A JSON object holding data about the new user, containing an ID, name and a favourite movie property. Example:
{id: 1, name: 'Kim', favouriteMovies: []}
Update user info (username) /users/[id] PUT A JSON object holding new data about the user to add structured like: {id: 1, name: 'Kim', favouriteMovies: []} A JSON object holding data about the updated user, containing an ID, name and a favourite movie property. Example:
{id: 1, name: 'Kim', favouriteMovies: []}
Add a movie to the list of favorites /users/[id]/[movieTitle] POST None A text message indicating the movie was successfully added to favorites
Remove a movie from the list of favorites /users/[id]/[movieTitle] DELETE None A text message indicating the movie was successfully removed of favorites
Delete user by ID /users/[id] DELETE None A text message indicating the user was successfully deleted
